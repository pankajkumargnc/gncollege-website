🛠️ CTO Suggestion: How to Split Large Files (Like AdminPanel.jsx)

If a file crosses 800-1000 lines, it becomes hard to manage. Here is the Step-by-Step strategy to split it:

  1. Create Sub-Folders: Create a folder like src/components/admin/.
  2. Extract Tabs into Components: Cut the code for specific tabs (e.g., Notices, Alerts) and paste them into separate files like AdminNoticesTab.jsx and AdminAlertsTab.jsx.
  3. Pass Props: If the child tab needs data (like notices array), pass it as a prop from the main AdminPanel: <AdminNoticesTab data={notices} />.
  4. Implement Lazy Loading: Inside AdminPanel.jsx, import them using React.lazy() so they only load when the user clicks on that tab, making the initial load lightning fast!

    const AdminNoticesTab = lazy(() => import('./admin/AdminNoticesTab'));
