⚠️ Firebase Storage Rules — Must Set This!
Image upload will fail without proper Storage rules. Go to Firebase Console → Storage → Rules → paste this:
FIREBASE STORAGE RULES
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read: if true;
      allow write: if true; // For testing — tighten later
    }
  }
}
