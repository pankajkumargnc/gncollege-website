## 🌟 Supreme Architecture Features

This is not a standard website; it is a **Custom Content Management System (CMS)** built from the ground up by Pankaj Kumar.

1. **Dynamic Quantum Routing:** Fully dynamic `App.jsx` handling nested structures, isolated static pages, and DB-driven custom URLs (`/p/:slug`).
2. **Advanced Glassmorphism UI:** Next-generation transparent blurring, animated aurora backgrounds, and glow-on-hover cursor tracking logic.
3. **Live Advanced Menu Builder:** An intelligent Drag & Drop style nested tree generator that builds the Main Navbar via Firebase in real-time.
4. **Quantum Diagnostic Engine:** A 40-phase ultra-scan system running in the Admin Panel that analyzes Memory Leaks, API Latency, Firebase Configs, and DOM Node toxicity.
5. **Smart Data Merging:** Intelligent fallback system that safely combines static legacy paths with modern cloud-driven data instantly.

## 🛠 Tech Stack Mastered

* **Frontend Engine:** React 18+ (Vite Bundler for sub-millisecond HMR)
* **Backend Matrix:** Google Firebase (Firestore Database, Rules, Storage emulation)
* **Animation Core:** Framer Motion principles, Custom Intersection Observers (`sa-up`, `sa-fade`), CSS Keyframes.
* **Component Structure:** Pure Functional Components with advanced hook implementations (`useMemo`, `useCallback`, `useRef`, Custom Draft Hooks).
